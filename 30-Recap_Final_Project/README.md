# Day 30: Final Project — Student Management System in C++ vs in Python

This folder contains C++ and Python implementations of a console-based Student Management System developed as the final project of the 30-Day C++ & Python learning challenge. The project integrates multiple programming concepts including data structures, object-oriented programming, file handling, and algorithmic thinking.


## Concept

The Student Management System demonstrates how fundamental and advanced programming concepts can be combined to build a functional application.  
It allows users to add, display, search, and delete student records while managing data using structured programming and object-oriented principles.
This project serves as a practical recap of the entire 30-day learning roadmap.


## C++ Implementation

- Implemented using classes and object-oriented programming  
- Uses Standard Template Library (STL) structures such as map, set, stack, and queue  
- Demonstrates inheritance and polymorphism  
- Includes file handling using fstream  
- Exception handling implemented using try/catch  
- Strong compile-time type checking and high performance  
- Suitable for performance-oriented and system-level applications  


## Python Implementation

- Implemented using classes and dynamic data structures  
- Uses dictionaries, sets, lists, and collections module  
- Demonstrates inheritance and method overriding  
- File handling implemented using built-in open() and JSON storage  
- Exception handling using try/except  
- Clean, readable, and concise syntax  
- Ideal for rapid development and scripting-based applications  


## Learning Objective

The objective of this project is to integrate all major programming concepts learned during the 30-day challenge and apply them in a practical system.
This project strengthens understanding of data organization, program structure, and object-oriented design in both C++ and Python.


## Contents

- C++_example.cpp — C++ implementation using STL and OOP  
- python_example.py — Python implementation using dictionaries and classes

## Summary

This final project consolidates the knowledge gained throughout the 30-Day C++ & Python learning roadmap.  
By implementing the same system in both languages, developers can compare structural programming in C++ with the flexibility and readability of Python.
C++ emphasizes performance, structure, and low-level control, while Python focuses on rapid development, simplicity, and developer productivity.  
Understanding both approaches provides a strong and versatile programming foundation.
