# Day 6: Advanced Functions C++ vs Python

This folder demonstrates advanced usage of functions in C++ and Python, focusing on flexibility, reusability, and clean function design.

## Concept

Advanced function features allow developers to write adaptable and maintainable code. They help reduce duplication, improve clarity, and support scalable program design.

## C++ Advanced Functions

- Function overloading (same function name, different parameter lists)
- Default arguments for optional parameters
- Inline functions for small, frequently used logic
- Pass-by-value vs pass-by-reference for performance control

## Python Advanced Functions

- Default arguments for optional parameters
- Keyword arguments for explicit and readable function calls
- Arbitrary arguments using `*args` and `**kwargs`
- Functions treated as first-class objects

## Learning Objective

Understand how advanced function features enhance modularity, readability, and flexibility in both C++ and Python.

## Folder Contents

- `cpp_example.cpp` — Advanced function features in C++
- `python_example.py` — Default, keyword, and arbitrary arguments in Python
